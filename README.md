🎬 Dashboard de Análise de Filmes - TMDB (1990–2017)







📖 Sobre o Projeto

Este projeto apresenta um Dashboard Interativo desenvolvido no Power BI utilizando dados do TMDB (The Movie Database), contendo informações sobre filmes lançados entre 1990 e 2017.

O objetivo foi transformar dados brutos em informações estratégicas por meio de técnicas de ETL, modelagem de dados, criação de métricas em DAX e visualizações interativas, permitindo explorar o comportamento da indústria cinematográfica ao longo dos anos.

Este projeto faz parte do meu portfólio de Análise de Dados e demonstra conhecimentos em Business Intelligence, tratamento de dados e desenvolvimento de dashboards.

🎯 Objetivos
Praticar processos de ETL utilizando Power Query;
Criar um modelo de dados organizado;
Desenvolver métricas utilizando DAX;
Construir visualizações intuitivas e interativas;
Gerar insights a partir de dados reais.
📊 Indicadores Analisados
🎥 Total de filmes lançados
⭐ Média das avaliações
💰 Receita total
💸 Orçamento total
📈 Lucro estimado
🔥 Filmes mais populares
🏆 Filmes mais bem avaliados
🎭 Distribuição por gênero
📅 Evolução dos lançamentos por ano
🌍 Comparação entre desempenho financeiro e avaliações
